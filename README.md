# ctf-kujira
Docker image and some tools for CTFer.

## Dependencies
* python3
* docker
* docker-compose
* bash or zsh

## How To Use
### Install (Linux)
```sh
$ git clone https://github.com/ctf-kujira/ctf-kujira kujira
$ cd kujira
$ python3 install
```
### Run
```sh
$ kujira
$ kujira --copy filename  # copy file or directory to volume before kujira run
```

### Uninstall (Linux)
```sh
$ cd kujira
$ python3 uninstall
```

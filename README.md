# ctf-kujira

Docker image and some tools for CTFer.

## Dependencies

- python3
- docker
- docker-compose
- bash or zsh

## How To Use

It works on only Linux.

### Install

```sh
$ git clone https://github.com/ctf-kujira/ctf-kujira kujira
$ cd kujira
$ python3 install
```

### Run

```sh
$ kujira # run bash
$ kujira COMMAND # run COMMAND
```

### Uninstall

```sh
$ cd kujira
$ python3 uninstall
```

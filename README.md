# ctf-kujira
Docker image for CTFer.

## How To Use
```sh
$ git clone https://github.com/akakou/ctf-kujira/
$ cd ctf-kujira
$ sudo docker-compose build
$ sudo docker-compose run ctf-kujira zsh
```
